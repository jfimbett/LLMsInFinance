## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/[instructor]/LLMsInFinance.git
   cd LLMsInFinance
   ```

2. **Create virtual environment**:
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Test your setup**:
   ```bash
   jupyter notebook
   ```
